# Change Log
[CCDatabasePeripheralLite](https://github.com/Iunius118/cc-dbp-lite#readme).
[Source](https://github.com/Iunius118/cc-dbp-lite).

## [Unreleased]
### Added
- New peripheral functions and database manipulation functions for Statement and ResultSet.
  Each function opens a connection to the database each time it is called and closes the connection after processing is complete.

### Removed
- All peripheral functions and database manipulation functions in v0.1.2 and earlier.

## [0.1.2]
### Fixed
- Fix memory leaks caused by remaining connections to databases.

## [0.1.1]
### Added
- Async execution functions and `dbstorage_response` event.
- Function `closeAll()`.

## [0.1.0]
- Initial release.
- Require Minecraft 1.20.1 and Forge 47.3.0.
- Require CC: Tweaked 1.20.1-forge-1.112.0.
